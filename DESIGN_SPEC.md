# Knowledge Extractor - Notion风格UI设计规范

## 🎨 设计理念

采用Notion的现代化设计语言：
- **简洁优雅** - 白底设计，充足的空白
- **高级排版** - 清晰的字体层级和间距
- **人性化** - 柔和的圆角和阴影
- **可视化** - 彩色图案和图标增强视觉吸引力

## 🎬 动画和过渡

### 过渡时间
- 快速: 150ms (按钮、小元素)
- 标准: 250ms (卡片、模态框)
- 缓慢: 350ms (页面过渡)

### 缓动函数
- 标准: `ease-out`
- 进入: `cubic-bezier(0.4, 0, 0.2, 1)`
- 退出: `cubic-bezier(0.4, 0, 1, 1)`

### 常见动画
- 按钮按下: `scale(0.98)` 150ms
- 卡片悬停: `translateY(-2px)` 250ms
- 模态框出现: `scale(0.95) → scale(1)` 250ms
- 淡入淡出: `opacity 0 → 1` 250ms

### 统计面板
- 大数字显示（高对比度）
- 下方显示标签
- 背景色浅灰
- 数字用主色突出

## 🎨 现代化设计细节

1. **充足的空白** - 不要让界面显得拥挤
2. **清晰的层级** - 通过大小、颜色、权重区分重要性
3. **微交互** - 按钮、卡片都有细微的反馈
4. **一致性** - 所有组件遵循统一的设计语言
5. **可访问性** - 足够的颜色对比度，清晰的焦点状态
